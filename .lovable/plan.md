

# خطة إضافة صفحة تعريب Pokémon Scarlet

## ملخص
إنشاء صفحة مستقلة لدعم تعريب Pokémon Scarlet مع محرر مخصص، تتبع نفس بنية صفحة Xenoblade الحالية ولكن بمعلومات وأدوات خاصة بلعبة بوكيمون.

## الملفات المطلوب إنشاؤها

### 1. صفحة الهبوط — `src/pages/Pokemon.tsx`
- تصميم Hero مشابه لصفحة Xenoblade مع ألوان بوكيمون (أحمر قرمزي `#E3350D`)
- شرح الخطوات الثلاث (رفع → معالجة → تحميل)
- استخدام مكون `GameInfoSection` الموجود مع بيانات بوكيمون:
  - **صيغة الملفات**: `.dat (FlatBuffers)` — ملفات النصوص تستخدم FlatBuffers داخل حاويات TRPAK
  - **الملفات المطلوبة**: ملفات النصوص المستخرجة من `common/` و `script/`، ملف القاموس
  - **الأدوات**: Trinity Mod Loader (gftool)، pkNX، FlatCrawler، GFMSG
  - **طريقة التعريب**: استخراج النصوص من TRPFS/TRPFD → تعديل FlatBuffers → إعادة الحزم
- أزرار: "ابدأ التعريب" → `/pokemon/process`، معلومات إضافية

### 2. صفحة المعالجة والمحرر — `src/pages/PokemonProcess.tsx`
- رفع ملفات النصوص (JSON dumps من sv-text أو ملفات .dat المستخرجة)
- دعم رفع ملفات JSON المستخرجة بأداة GFMSG أو pkNX (صيغة key=value أو JSON)
- استخراج النصوص وعرضها في المحرر المدمج (نفس مكون `Editor.tsx` الحالي مع تخصيص `game="pokemon"`)
- أدوات خاصة:
  - فحص القاموس والتطابق
  - ترجمة تلقائية بالذكاء الاصطناعي
  - تحسين الصياغة
  - تصدير النتائج كملف JSON جاهز

### 3. صورة خلفية — `src/assets/pokemon-sv-hero-bg.jpg`
- استخدام صورة placeholder أو تدرج لوني بأسلوب بوكيمون

### 4. تحديث الملفات الموجودة

**`src/App.tsx`** — إضافة المسارات:
```
/pokemon → Pokemon landing
/pokemon/process → PokemonProcess
```

**`src/components/editor/types.tsx`** — إضافة دعم `game: "pokemon"` في التصنيفات

**الصفحة الرئيسية `Xenoblade.tsx`** — إضافة رابط لصفحة بوكيمون (اختياري)

## معلومات تقنية عن Pokémon Scarlet

| العنصر | التفاصيل |
|--------|----------|
| محرك اللعبة | Trinity Engine (Game Freak) |
| صيغة الأرشيف | TRPFS/TRPFD |
| صيغة الحزم | TRPAK |
| صيغة النصوص | FlatBuffers (.dat/.bin) |
| مجلدات النصوص | `common/` (واجهة/عام) و `script/` (حوارات) |
| أدوات الاستخراج | gftool/Trinity Mod Loader, pkNX, SCVI_Extract |
| أداة تحرير النصوص | GFMSG (C# library + GUI) |

## المحرر المخصص — الميزات
- **محلل JSON مدمج**: قراءة ملفات النصوص المستخرجة (JSON format مع labels وhashes)
- **قاموس بوكيمون عربي**: دعم تحميل قاموس مصطلحات (أسماء البوكيمون، الهجمات، القدرات)
- **ترجمة ذكية بالسياق**: تمييز تلقائي لنوع المحتوى (أسماء بوكيمون، هجمات، واجهة، حوارات)
- **معالجة عربية**: نفس خط المعالجة الحالي (Reshaping + BiDi) مع تصدير
- **تصدير JSON**: إعادة تصدير الترجمات بنفس الصيغة الأصلية
